1、函数之间的关系是平行的，是独立的，在函数定义时不能嵌套定义，
    即一个函数的定义的函数体内不能包含另外一个函数的完整定义。
    但是可以在进行嵌套调用，即在调用一个函数的过程中调用另外一个函数。

2、函数递归调用
    在调用一个函数的时候，又直接或间接调用了该函数本身。
    又分为 直接递归：在函数处理过程又直接调用了自己。
           简介递归：p() 调用 q()，而 q() 反过来调用 p()。

3、内部函数和外部函数
    内部函数定义：
            static  类型说明符  函数名([形参表])
        只能被本文件中的函数调用，在其他文件不能调用此函数。
        内部函数也称静态函数，但此处的 static 的含义不是指存储方式而是指对函数的
        调用只局限于本文件。
    外部函数定义：
            extern  类型说明符  函数名([形参表])
        因为函数与函数之间都是并列的，函数不能嵌套定义，所以函数在本质上都具有
        外部性质。因此在定义函数时省去了 extern 说明符时，则隐含为外部函数。
        定义为外部函数，不仅可被定义它的源文件调用，而且可以被其他的文件中的函数
        调用，即其作用范围不只局限于本源文件，而且是整个程序的所有文件。

4、main 函数
    C 程序执行总是从 main() 开始。
    main() 函数可以调用其他函数，其他函数不能调用 main()， main() 不能调用自己。
    main() 可以带有两个参数：
        int main(int argc, char *argv[])
        {
            函数体
        }
        形参 argc 表示传给程序的参数个数，其值至少为 1，
        而 argv 则是指向字符串的指针数组。
